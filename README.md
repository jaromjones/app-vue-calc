# my-frist-vue-app

> My frist! Vue.js project

## Build Setup

``` bash
# install dependencies
npm install

# build for development
npm run build

# build for production w/ minification
npm run deploy

# serve with hot reload @ localhost:8080
npm run dev

# clean up dependencies (requires git)
npm run clean
```
